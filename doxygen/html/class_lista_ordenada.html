Detailed Description

template<typename T>
class ListaOrdenada< T >

Plantilla de clase que representa una lista cuyos elementos se mantienen ordenados ascendentemente al momento de la inserción. Utiliza el principio de composición incluyendo un objeto Lista doblemente enlazada internamente.

Template Parameters
TTipo de dato que almacenarán los elementos. Debe soportar operadores relacionales (como < y <=).
